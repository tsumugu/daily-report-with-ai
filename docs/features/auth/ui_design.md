# ユーザー認証機能 UI/UX設計書（UI Design）

## 1. 概要
ログイン・サインアップ・ログアウト画面のUI/UX設計方針を定義する。

---

## 2. ログイン画面

### 画面構成
- ロゴ or サービス名
- 入力フォーム（メール / パスワード）
- ログインボタン
- サインアップ画面へのリンク

### UIコンポーネント
| 項目 | コンポーネント | 備考 |
|:-----|:--------------|:-----|
| メール | Input (email) | 必須 |
| パスワード | Input (password) | 必須、表示切替ボタン |
| ログイン | Button (Primary) | 「ログイン」 |
| サインアップリンク | Link | 「アカウントを作成」 |

### ワイヤーフレーム（イメージ）

```
┌──────────────────────────┐
│                          │
│       [ Logo ]           │
│                          │
│  メールアドレス          │
│  ┌────────────────────┐  │
│  │                    │  │
│  └────────────────────┘  │
│  パスワード              │
│  ┌────────────────────┐  │
│  │ ●●●●●●●●     👁   │  │
│  └────────────────────┘  │
│                          │
│      [ ログイン ]        │
│                          │
│  アカウントを作成 →      │
└──────────────────────────┘
```

## 3. サインアップ画面

### 画面構成
- ロゴ or サービス名
- 入力フォーム（メール / パスワード / 確認用パスワード）
- 登録ボタン
- ログイン画面へのリンク

### UIコンポーネント
| 項目 | コンポーネント | 備考 |
|:-----|:--------------|:-----|
| メール | Input (email) | 必須 |
| パスワード | Input (password) | 必須、8文字以上 |
| パスワード確認 | Input (password) | 必須 |
| 登録 | Button (Primary) | 「アカウントを作成」 |
| ログインリンク | Link | 「ログインはこちら」 |

---

## 4. ログアウト
- ヘッダーメニュー内に「ログアウト」リンク/ボタン配置
- 実行後はログイン画面へリダイレクト

---

## 5. レイアウト方針
- モバイルファースト
- 中央寄せ・シンプルなフォームUI
- 余白を十分に取り、入力しやすさを重視

## 6. インタラクション
- バリデーションエラー時はフィールド下に赤字で表示
- ログイン成功時は日報一覧へ遷移
- サインアップ成功時はログイン画面 or 自動ログイン

## 7. デザイントークン
- Shadow-CN / Tailwind CSSベース
- デザインシステム準拠
