## 1. 概要（Why）

### 背景

現在の日報一覧では、日付・できごと（冒頭50文字）・よかったこと/改善点の有無アイコンのみが表示されている。ユーザーが一覧から日報を選ぶ際、より多くの情報が表示されると、振り返りや分析の効率が向上する。

### 目的

- 日報一覧でより多くの情報を一目で確認できるようにする
- よかったこと・改善点の進捗状況を一覧で把握できるようにする
- 日報選択の判断材料を増やす

---

## 2. ユーザーストーリー

### US-1: 一覧での情報確認

```
As a ユーザー
I want to 日報一覧でよかったこと・改善点の詳細情報を確認したい
So that どの日報を詳しく見るべきか判断できる
```

### US-2: 進捗状況の把握

```
As a ユーザー
I want to 一覧でよかったこと・改善点の進捗状況を確認したい
So that 振り返りの優先順位を決められる
```

---

## 3. 機能要件

### 3.1 追加表示情報（Phase 2 v2）

| 項目                         | 説明                                     | 優先度 |
| :--------------------------- | :--------------------------------------- | :----- |
| よかったことの数             | その日報に含まれるよかったことの件数     | 高     |
| 改善点の数                   | その日報に含まれる改善点の件数           | 高     |
| よかったことのステータス概要 | 再現成功・定着しているものの数           | 中     |
| 改善点のステータス概要       | 完了・習慣化しているものの数             | 中     |
| できごとの全文表示           | 現在は冒頭50文字のみ、全文表示オプション | 低     |

### 3.2 表示形式

#### 現在の表示

- 日付
- できごと（冒頭50文字）
- よかったこと有無アイコン（ハート）
- 改善点有無アイコン（ファイル）

#### 追加後の表示（案）

```
┌─────────────────────────────────────┐
│ 2025/12/10（火）                     │
│ 💚 よかったこと: 2件                  │
│    ✓ 再現成功: 1件  ✓ 定着: 1件     │
│ 📄 改善点: 1件                        │
│    ✓ 完了: 1件                        │
│ ─────────────────────────────────── │
│ チーム会議で積極的に発言できた...    │
└─────────────────────────────────────┘
```

### 3.3 データ取得

- 日報一覧APIで、よかったこと・改善点の詳細情報も取得する
- 現在はIDのみ取得しているが、ステータス・成功回数も含める

---

## 4. 非機能要件

- パフォーマンス: 一覧表示のパフォーマンスを維持（1秒以内）
- レスポンシブ: モバイル/PC両対応
- データ量: 一覧取得時のデータ量増加を最小限に

---

## 5. 成功基準

- ユーザーが一覧から日報を選ぶ際の判断材料が増える
- よかったこと・改善点の進捗状況を一覧で把握できる

---

## 6. エッジケース

| ケース                    | 対応                                       |
| :------------------------ | :----------------------------------------- |
| よかったこと・改善点が0件 | 「よかったこと: 0件」「改善点: 0件」と表示 |
| ステータスが未設定        | ステータス概要は表示しない                 |
| データ取得エラー          | 既存の情報のみ表示（エラー時は現状維持）   |

---

## 7. 将来の拡張（Phase 3以降）

### AI分析機能

| 項目           | 説明                                      |
| :------------- | :---------------------------------------- |
| AIさまり       | 日報内容をAIが分析し、サマリーを生成      |
